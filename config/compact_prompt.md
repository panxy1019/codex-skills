Summarize the conversation for continuation by another Codex session.

Preserve:
- the user's current objective and exact constraints
- important decisions and rationale
- files, directories, repositories, and URLs that matter
- commands run and their meaningful outcomes
- code changes already made
- validation performed and remaining verification gaps
- blockers, open questions, and the next best action

Remove:
- repeated discussion
- routine tool noise
- failed paths that do not affect the next action
- verbose logs unless a specific line matters
- any secrets, passwords, tokens, private keys, or credentials

Format:

# Compacted Handoff

## Goal

## Current State

## Decisions

## Files And Artifacts

## Commands And Checks

## Open Questions

## Next Step
